Discover Bourbon

Discover Bourbon is a recommendation and informational site about all things bourbon. Users can log into the website, leave reviews, and get recommendations based on their reviews. Users can also access an information page about each bourbon that describes the bourbon (i.e. proof, age, mash bill, distillery, etc.) as well as similar bourbons within and outside the same distillery. The website can be accessed and used to its full potential at discoverbourbon.herokuapp.com. The website is missing many bourbons on market and is still under some finalization.

This website was built using python, Django, HTML/CSS, sklearn, and is hosting on Heroku's VPS.
